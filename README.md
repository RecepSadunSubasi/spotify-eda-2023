This project presents an exploratory data analysis (EDA) of the most streamed Spotify songs in 2023.

##  Dataset Overview
- 953 songs
- 24 variables
- Includes popularity metrics and musical features

##  Analysis Performed
- Data type inspection and correction
- Missing value analysis
- Top streamed songs and artists
- Playlist visibility vs stream relationship

##  Tools Used
- Python
- Pandas
- Jupyter Notebook

##  Key Insights
- A small number of artists dominate total streams.
- Songs appearing in many playlists tend to have higher average streams.
- Dataset required data type correction for accurate aggregation.
